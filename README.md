# teleop_joy
How to control robot using a joystick.

------

## Built with

- ROS Noetic under Ubuntu 20.04 LTS

------

## Getting Started

### Installation

``` $ sudo apt-get install ros-noetic-joy```
    
------

Run teleop_joy launch.


``` bash
$ sudo chmod a+rw /dev/input/js0
```

``` bash
$ roslaunch teleop_joy teleop_joy.launch
```
